IAR Build Tools repository

This is IAR third-party repository, which we use for hosting our own Ubuntu packages.

We currently support Ubuntu 22.04 on our repository. The repository supports x86 (64-bit only).

  1. Install the following Ubuntu packages:

              sudo apt update
          sudo apt install -y curl apt-transport-https

  2. Obtain a copy of our signining key:

            curl https://felipe-iar.github.io/apt/iar.asc | gpg --dearmor > iar.gpg
        sudo install -o root -g root -m 644 iar.gpg /etc/apt/trusted.gpg.d/

  3. Add the repository to your sources list and update:

              sudo sh -c 'echo "deb [arch=amd64] https://felipe-iar.github.io/apt/stable stable main" > /etc/apt/sources.list.d/iar.list'
          sudo apt-get update

Now you can install any package of our APT repository. As an example, try installing the bxarm package:

          sudo apt-get install bxarm
          lightlicensemanager setup 0123-456-654-3210 user@mail.com
          iccarm --version
