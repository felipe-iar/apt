IAR Build Tools repository

This is IAR third-party repository, which we use for hosting our own Ubuntu packages.

We currently support Ubuntu 22.04 on our repository. The repository supports x86 (64-bit only).

  1. If you are using a minimal Ubuntu image or a Docker image, you may need to install the following packages:

    sudo apt update
sudo apt install -y curl apt-transport-https
curl https://felipe-iar.github.io/apt/iar.asc | gpg --dearmor > iar.gpg
sudo install -o root -g root -m 644 iar.gpg /etc/apt/trusted.gpg.d/
sudo sh -c 'echo "deb [arch=amd64] https://felipe-iar.github.io/apt jammy main" > /etc/apt/sources.list.d/iar.list'
sudo apt-get update
sudo apt-get install bxarm
iccarm --version
